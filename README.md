tmx
===

This package provides support for reading [tmx][1] (Tile Map XML) files, used by
the map editor [Tiled][].

[1]: https://github.com/bjorn/tiled/wiki/TMX-Map-Format
[Tiled]: https://github.com/bjorn/tiled/
